# Source Notes

These are the factual sources used to build the eval. Update this file if product specs or the eval premise changes.

## Claude skill structure

- Anthropic engineering article: Agent Skills are directories with a `SKILL.md` file; `SKILL.md` begins with YAML frontmatter containing `name` and `description`; skills can include additional referenced files.
  - https://www.anthropic.com/engineering/equipping-agents-for-the-real-world-with-agent-skills
- Anthropic Claude Code best practices: project skills can be placed under `.claude/skills/<skill-name>/SKILL.md` and invoked directly or used automatically when relevant.
  - https://www.anthropic.com/engineering/claude-code-best-practices

## MTF/lpmm references

- LensTip Sony FE 16-35mm f/2.8 GM II image resolution: high/even center performance, over 60 lp/mm at f/2.8, no weak spots at full-frame edge, about 50 lp/mm or better at longer focal lengths from f/2.8, almost 59 lp/mm at 16mm f/2.8 edge.
  - https://www.lenstip.com/654.4-Lens_review-Sony_FE_16-35_mm_f_2.8_GM_II_Image_resolution.html
- LensTip Sony FE 40mm f/2.5 G image resolution: almost 65 lp/mm center wide open, above 75 lp/mm stopped down, full-frame edge below decency at f/2.5/f/2.8 but improves by about f/3.5.
  - https://www.lenstip.com/602.4-Lens_review-Sony_FE_40_mm_f_2.5_G_Image_resolution.html
- LensTip Sony FE 35mm f/1.8 image resolution: center strong, full-frame edge unacceptable wide open, decent around f/2.0-f/2.2, peak a little over 50 lp/mm at full-frame edge.
  - https://www.lenstip.com/593.4-Lens_review-Sony_FE_35_mm_f_1.8_Image_resolution.html
- LensTip Sigma 18-35mm f/1.8 DC HSM image resolution: excellent APS-C performance, about 48 lp/mm at 18-24mm wide open, record-level 55 lp/mm stopped down, edge good; still APS-C and not a full-frame a7R VI travel solution.
  - https://www.lenstip.com/374.4-Lens_review-Sigma_A_18-35_mm_f_1.8_DC_HSM__Image_resolution.html
